# jQuery发布4.0版本

2006 年 1 月 14 日，John Resig 在纽约市的 BarCamp 大会上发布了名为 jQuery 的 JavaScript 库。如今，20 年过去了，jQuery 团队很高兴地宣布 jQuery 4.0.0 正式版发布。经过漫长的开发周期和多次预发布，jQuery 4.0.0 带来了诸多改进和现代化。这是近 10 年来首个主要版本，其中包含一些不兼容的变更，不过，预计大多数用户只需对现有代码进行少量修改即可完成升级。

* jQuery 4.0 不再支持 IE 10 及更早版本
* jQuery 4.0 增加了对Trusted Types的支持
* jQuery 4.0 源代码已迁移为 ES 模块
* jQuery 4.0 删除了以下 API
    * `jQuery.isArray`
    * `jQuery.parseJSON`
    * `jQuery.trim`
    * `jQuery.type`
    * `jQuery.now`
    * `jQuery.isNumeric`
    * `jQuery.isFunction`
    * `jQuery.isWindow`
    * `jQuery.camelCase`
    * `jQuery.nodeName`
    * `jQuery.cssNumber`
    * `jQuery.cssProps`
    * `jQuery.fx.interval`

更多信息请查看：https://blog.jquery.com/2026/01/17/jquery-4-0-0/
