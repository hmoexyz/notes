threading.local() жҳҜ Python ж ҮеҮҶеә“жҸҗдҫӣзҡ„гҖҢзәҝзЁӢеұҖйғЁеӯҳеӮЁпјҲThread Local Storage, TLSпјүгҖҚж–№жЎҲпјҢи®©еҗҢдёҖж®өд»Јз ҒеңЁдёҚеҗҢзәҝзЁӢйҮҢжӢҘжңүеҗ„иҮӘзӢ¬з«Ӣзҡ„еҸҳйҮҸз©әй—ҙпјҢд»ҺиҖҢйҒҝе…ҚеҠ й”ҒпјҢд№ҹйҒҝе…ҚдәҶеұӮеұӮдј еҸӮзҡ„зӢјзӢҲгҖӮ

1. дёәд»Җд№ҲйңҖиҰҒзәҝзЁӢеұҖйғЁеӯҳеӮЁпјҹ

еңЁеӨҡзәҝзЁӢзҺҜеўғдёӢпјҢеҰӮжһңеӨҡдёӘзәҝзЁӢе…ұдә«еҗҢдёҖдёӘе…ЁеұҖеҸҳйҮҸпјҢе°ұеҝ…йЎ»пјҡ

  1. еҠ й”Ғ вҶ’ д»Јз ҒеҸҳеӨҚжқӮгҖҒжҖ§иғҪдёӢйҷҚпјӣ
  2. жҲ–иҖ…еұӮеұӮдј еҸӮ вҶ’ д»Јз ҒиҮғиӮҝгҖҒеҸҜз»ҙжҠӨжҖ§е·®гҖӮ

жңүдәӣеңәжҷҜеҸӘжғіи®©зәҝзЁӢеҗ„иҮӘжҢҒжңүдёҖд»ҪеүҜжң¬пјҢдә’дёҚе№Іжү°пјҡ

  • Web жңҚеҠЎпјҡжҜҸдёӘиҜ·жұӮзәҝзЁӢз»‘е®ҡиҮӘе·ұзҡ„ user_idгҖҒdb_connпјӣ
  • ж—Ҙеҝ—пјҡжү“еҚ°зәҝзЁӢеҗҚ + иҜ·жұӮ IDпјҢж–№дҫҝй“ҫи·ҜиҝҪиёӘпјӣ
  • ж•°жҚ®еә“иҝһжҺҘжұ пјҡзәҝзЁӢеӨҚз”ЁиҝһжҺҘпјҢдҪҶиҝһжҺҘжң¬иә«дёҚи·ЁзәҝзЁӢдј йҖ’гҖӮ

иҝҷж—¶ TLS е°ұжҳҜжңҖдјҳи§ЈгҖӮ

2. threading.local() жҳҜд»Җд№Ҳпјҹ

threading.local() иҝ”еӣһдёҖдёӘгҖҢйӯ”жі•еҜ№иұЎгҖҚпјҡ
еҜ№е®ғзҡ„еұһжҖ§иөӢеҖјпјҢеҸӘдјҡеңЁеҪ“еүҚзәҝзЁӢеҸҜи§Ғпјӣе…¶е®ғзәҝзЁӢзңӢдёҚеҲ°гҖҒж”№дёҚеҲ°гҖӮ

1import threading
2
3tls = threading.local()   # 1. еҲӣе»ә TLS еҜ№иұЎ
4
5def worker(idx):
6    tls.value = idx       # 2. еҗ„зәҝзЁӢеҶҷиҮӘе·ұзҡ„еҖј
7    print(f'Thread {idx} sees {tls.value}')
8
9for i in range(5):
10    threading.Thread(target=worker, args=(i,)).start()
11

иҫ“еҮәпјҲйЎәеәҸеҸҜиғҪдёҚеҗҢпјүпјҡ

1Thread 0 sees 0
2Thread 4 sees 4
3Thread 1 sees 1
4Thread 2 sees 2
5Thread 3 sees 3
6

3. еҶ…йғЁеҺҹзҗҶпјҡз»ҝзӣ’еӯҗйҮҢзҡ„еӯ—е…ё

CPython е®һзҺ°йҮҢпјҢжҜҸдёӘзәҝзЁӢеҜ№иұЎпјҲthreading.Thread зҡ„еә•еұӮ PyThreadStateпјүйғҪз»ҙжҠӨдёҖдёӘз§Ғжңүеӯ—е…ёгҖӮ
tls.xxx = value зҡ„жң¬иҙЁжҳҜпјҡ

1# дјӘд»Јз Ғ
2current_thread_dict[id(tls)]['xxx'] = value
3

id(tls) дҪңдёә key дҝқиҜҒдёҚеҗҢ local() е®һдҫӢд№Ӣй—ҙдә’дёҚе№Іжү°пјӣ
еҪ“еүҚзәҝзЁӢеӯ—е…ёдҝқиҜҒзәҝзЁӢд№Ӣй—ҙдә’дёҚе№Іжү°гҖӮ

6. еёёи§Ғеқ‘ & жіЁж„ҸдәӢйЎ№

еқ‘зӮ№иҜҙжҳҺ
зәҝзЁӢжұ /еҚҸзЁӢж··з”Ёthreading.local еҸӘеңЁеҺҹз”ҹзәҝзЁӢйҡ”зҰ»пјҢеҚҸзЁӢжҲ–зәҝзЁӢжұ еӨҚз”ЁзәҝзЁӢж—¶дјҡеҮәзҺ°гҖҢж•°жҚ®дёІеҸ°гҖҚгҖӮPython 3.7+ иҜ·дјҳе…Ҳз”Ё contextvarsгҖӮ
дёҚиғҪи·ЁзәҝзЁӢдј йҖ’еӯҗзәҝзЁӢж— жі•и®ҝй—®зҲ¶зәҝзЁӢи®ҫзҪ®зҡ„еҖјпјӣйңҖиҰҒжҳҫејҸдј еҸӮжҲ–йҳҹеҲ—гҖӮ
еҶ…еӯҳжі„жјҸзәҝзЁӢз»“жқҹдҪҶ TLS йҮҢзҡ„еҜ№иұЎиӢҘеҫӘзҺҜеј•з”ЁпјҢеҸҜиғҪе»¶иҝҹйҮҠж”ҫгҖӮе»әи®®еңЁзәҝзЁӢж”¶е°ҫжүӢеҠЁ del tls.xxxгҖӮ
з»§жүҝеӨұж•ҲиҮӘе®ҡд№ү Thread еӯҗзұ»ж—¶пјҢеҲ«еҝҳдәҶи°ғз”Ё super().__init__()пјҢеҗҰеҲҷ TLS еҲқе§ӢеҢ–дјҡејӮеёёгҖӮ

1import threading, random, time
2
3local = threading.local()
4
